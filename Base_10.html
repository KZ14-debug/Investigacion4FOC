Base 8

El proceso de cambio a base 8 es el siguiente:

  1. Se convierte 123456789 a base 2.

    123456789 = 111010110111100110100010101

  2. Se separa en grupos de 3 bits.

    111 010 110 111 100 110 100 010 101

  3. Se convierte cada grupo de 3 bits a base 8.

    111 =7
    010 =2
    110 =6
    111 =7
    100 =4
    110 =6
    100 =4
    010 =2
    101 =5

  4. Se une cada digito obtenido.

    7 2 6 7 4 6 4 2 5 = 726746425

El resultado de cambio de base es (123456789)10 = (726746425)8

  2. Se separa en grupos de 4 bits.
